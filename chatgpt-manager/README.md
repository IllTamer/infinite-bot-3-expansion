# Infinite Bot 3.1.0 - ChatGPTManager

## 使用

向 chatgpt 进行提问: 输入 prefix+问题 

> 提问后将自动开启一段对话 (conversation)，并在一段时间未收到回复后过期

## Notice

- 附属自带作者申请的 token 链接作为默认设置，并定期更新。若您有自定义模型的需求，可自行替换。
